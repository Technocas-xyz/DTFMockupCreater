# T-Shirt Images

Place your actual t-shirt photos here with the following naming convention:

```
{color-name}-{front|back}.png
```

## Examples:
- `black-front.png`
- `black-back.png`
- `navy-front.png`
- `navy-back.png`
- `white-front.png`
- `white-back.png`
- `dark-heather-front.png`
- `charcoal-front.png`
- `forest-green-front.png`
- `royal-blue-front.png`

## Color names (must match exactly):
- black
- navy
- dark-heather
- charcoal
- dark-grey
- forest-green
- royal-blue
- light-blue
- white
- sand
- maroon
- cardinal-red
- red
- orange
- pink

## Image specs:
- Format: PNG with transparent background
- Recommended size: 700x850px (or proportional)
- The t-shirt should fill most of the image
- Ensure the chest area is centered

If no image is found for a color, the app will auto-generate a realistic t-shirt graphic.
